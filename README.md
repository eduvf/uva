# 🍇 uva markup

## headers

up to 3 levels

```
# h1
## h2
### h3
```

## paragraphs

are separated by empty lines

```
this is paragraph 1
this is still paragraph 1

this is paragraph 2
```

you can force line breaks with 2 or more spaces at the end of the line

## code

use three backticks \`\`\`

```
some code
```

## lists

can't be nested

### example

```
* item 1
* item 2
```

* item 1
* item 2

## links & content

follow Markdown syntax, but have to be on their own line

### example

```
[eduvf](https://eduvf.com)
![](https://upload.wikimedia.org/wikipedia/commons/b/b3/Grapes-3550742_960_720.jpg)
```
